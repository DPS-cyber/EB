# EduBot

EduBot is a simple web-based educational assistant built with HTML, JavaScript, and CSS. It helps you learn and manage tasks with interactive chat and useful features.

---

## Features

- 🕒 Tell time  
- 🌦️ Get weather info  
- 🎯 Manage goals  
- 📅 Schedule panel  
- 🤖 Chat & assist  
- 🔍 Google search fallback
